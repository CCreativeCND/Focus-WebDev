Web Developers

What does a Web Developer do?

A web developer creates web pages and web-based applications using:

  • HTML, Javascript, and CSS
  • PHP, Microsoft .Net Visual Basic, or Java
  • MySQL, Microsoft SQL Server, IBM and Oracle

Career Pathways

There are 2 main pathways:

  • Front-End
  • Back-End

You can look at the links above to see the pathways.

Education

A web developer needs at least a bachelor's degree in computer science or a related field.

Salary

A web developer makes about $80,000 a year, and about $33.38 an hour.
